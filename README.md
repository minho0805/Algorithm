#  ⛓️Algorithm

이 저장소는 [BaekjoonHub](https://github.com/BaekjoonHub/BaekjoonHub)를 통해 알고리즘 문제 풀이를 자동 업로드하는 공간입니다.  
백준에서 푼 문제들이 자동으로 commit & push 되어 기록됩니다.  

---

## 📂 Repository 구조
- `/백준 단계별/문제번호_문제이름`  
- `/프로그래머스/문제이름` 
- `/LeetCode/문제번호.문제이름`

---

## ⚙️ 사용 방법
1. 크롬 브라우저에서 [BaekjoonHub 확장 프로그램](https://chrome.google.com/webstore/detail/baekjoonhub) 설치
2. GitHub 레포지토리와 연동
3. (백준/프로그래머스/리트코드)에서 문제 풀이 → 제출 성공 → 자동으로 이 레포지토리에 push
